# armhf-proot-for-aarch64-android
Anlinux armhf proot for aarch64 android using Termux.


###Steps√∑

1- mkdir ubuntu (create directory to contain the proot)

2- wget https://github.com/EXALAB/Anlinux-Resources/blob/master/Rootfs/Ubuntu/armhf/ubuntu-rootfs-armhf.tar.xz
(Download the proot)

3- bash ubuntu.sh (decompress the rootfs and installs it)
